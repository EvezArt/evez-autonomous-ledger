# EVEZ Autonomous Ledger — Self-Auditing Transaction Log

Every EVEZ action, every agent decision, every financial flow — tracked, auditable, immutable.

## Philosophy
In a system of autonomous agents, the ledger IS the trust layer. If it's not in the ledger, it didn't happen.

## Quick Start

```bash
git clone https://github.com/EvezArt/evez-autonomous-ledger.git
cd evez-autonomous-ledger
pip install -r requirements.txt
python ledger.py
```

## Features
- Append-only transaction log
- SHA-256 hash chaining
- CAIN contradiction detection
- MAES event integration
- Export to JSON/CSV for external audit
